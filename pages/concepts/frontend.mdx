#  Frontend Overview
## How to edit the Frontend of the Sokrates Platform 
This Documentatione explains the structure of the Frontend of the Sokrates Platform and where to find the relevant code to make changes in The UI.
The Importand Folders For the Frontend are The apps/web/app and the apps/web/components folder:
- **apps/web/app**: This folder contains the main application code, including the Main pages, layouts, and global styles.
- **apps/web/components**: This folder contains reusable UI components like Buttons used throughout the application, as well as specific subpages grouped by components.

### Components

The reusable components used throughout the Sokrates Platform can be found in the apps/web/components folder.

#### UI Components
In this folder you will find all the reusable UI components from th shadcn/UI library, All the components are are stlyed according to the Sokrates Platform Design which can be found on the figma board.
New Components should always be implemented using the unifomly designed shadcn UI components as a base.

-**Specific Components**
Some of the components include Buttons, Modals, Forms, and Navigation elements. These components are designed to be flexible and can be easily integrated into different parts of the application.
