# Setting up a Dev environment

This is a guide to set up a dev environment for LearnHouse, it will guide you through the installation of the backend and the frontend.

## Prerequisites

- [Docker](https://docs.docker.com/get-docker/)
- [Node.js 21.x](https://nodejs.org/en/download/)
- [pnpm](https://pnpm.io/installation)
- [Python 3.12.x](https://www.python.org/downloads/)
- [Poetry](https://python-poetry.org/docs/)
- macOS, Linux or Windows

## Backend configuration

<Steps>

### Get the repository

```shell copy
git clone https://github.com/sokrates-ai/platform
```

```shell copy
cd platform
```

### Prepare Local Development Environment

```shell copy
make setup
```

This will install all the dependencies needed and spin up the following containers for development:

- A **Postgres** database
- A **Redis** server
- A **Chromadb** server

<Callout type="warning">
  Make sure your container runtime has at least 4GB of RAM allocated, as default
  settings may vary by tool
</Callout>

### Run the Backend and Frontend in Development Mode

#### Run the Backend in Dev mode

```shell copy
make api-dev
```

```shell copy
make web-dev
```

#### Check the API

Go to [http://localhost:1338/docs/](http://localhost:1338/docs/)

</Steps>
